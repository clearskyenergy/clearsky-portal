API keys & data sources

Code basis: NEC 2023 Table 310.16 ampacity, 310.15(B)(1) ambient correction, 310.15(C)(1) bundling, 110.14(C) termination limits, 240.4(D) small-conductor OCPD caps, and the 125% continuous-load rule. Output is an engineering aid — the EOR stamps the final design.

Install conditions

Termination column caps the usable ampacity per 110.14(C): even 90°C wire is limited to the terminal's rating for final sizing. Derates use the insulation column, then we compare.

Feeder runs

Enter each run's continuous load (amps) and one-way length (ft). Non-continuous loads: uncheck "continuous" to skip the 125% factor. Conductor count sets bundling derate.

Transformer sizing (optional)

Recommends the next standard kVA rating above (connected × demand × (1+growth)). Standard sizes: 15, 30, 45, 75, 112.5, 150, 225, 300, 500, 750, 1000, 1500, 2000, 2500 kVA.
