orgId Backfill

One-time tool: stamps every existing project in Firestore with an orgId so it stays visible under the new tenant security rules. Run it once, confirm the count, then delete this file from the repo.

Before you run: sign in with your ClearSky admin Google account (the same one that owns the existing projects). This writes an orgId to projects that don't have one yet. It never overwrites an orgId that already exists.
